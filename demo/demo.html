Upload a file

To use this page, host a local webserver with the root directory set as the one containing this file.

Your image will be accessible through the img variable in the console.

Code example:

	var myResource = new VtfImageResource([
		new Frame( img )
	]);
	var myVTF = new Vtf( [img.width, img.height], [myResource], 'RGBA8888' );
	saveAs( myVTF.blob(), 'myVTF.vtf' );
